## 项目介绍

**高仿ReadHub，项目中整合 vue + node + mongodb + express + axios  + vue-router，一个前端全栈简单Demo**

- [x] ReadHub热门话题

- [x] ReadHub科技动态

- [x] ReadHub开发者咨询
